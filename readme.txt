A plugin to add widgets created with nativery

== Description ==

This plugin allows you to add to your blog widgets created with <a href="http://www.nativery.com">nativery.com</a>.


= Including: =
* ability to add nativery widgets in sidebar, before and after the content


== Installation ==

= Install Via Plugin Repository =
1. Log into WordPress admin for your site
2. Go to 'Plugins -> Add New' and choose 'Search'
3. Search for nativery and click 'Install Now'
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Adjust settings


= Install Via ftp =
1. Download the zip file 
2. Upload Nativery Plugin to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Adjust settings

== Remove plugin ==

1. Deactivate plugin through the 'Plugins' menu in WordPress
2. Delete plugin through the 'Plugins' menu in WordPress
